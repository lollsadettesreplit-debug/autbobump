# Discord Utility Service

**Educational project for understanding Discord API automation**

## ⚠️ Important Notice
This project is for **EDUCATIONAL PURPOSES ONLY**. Automating user accounts violates Discord's Terms of Service and may result in account termination.

## Features
- Demonstrates Discord API interactions
- Web server for health monitoring
- Educational code structure

## Setup
1. Clone repository
2. Install dependencies: `npm install`
3. Set environment variables
4. Run: `npm start`

## Environment Variables
- `USER_TOKEN`: Discord user token (for educational purposes)
